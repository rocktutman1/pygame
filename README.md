# Guide to using epsteingame

## Description

The **epstein game** is a pygame game

## Requirements

- **Python3** or higher
- *pygame*, *sys*, *random* modules installed

## Instillation 

Clone the git repository

> git clone https://github.com/rocktutman1/pygame


## Usage

Just run the game with
>python3 game.py
or on windows
>python game.py

## Playing the game

Move around with the arrow keys, after beating the game or dying hit any key to exit

## To change the player sprite

Just replace the epstein.jpeg file with any other jpeg named epstein.jpeg

No configuration is curently available
